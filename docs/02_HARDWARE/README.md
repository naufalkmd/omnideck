# Hardware Docs Scope

This folder contains implementation-ready hardware specs.

## Owns

- `HARDWARE_DESIGN.md` — board architecture and physical design
- `MODULE_INTERFACE_SPEC.md` — electrical interfaces/signals
- `MOTHERBOARD_V1_SPEC.md` — motherboard v1 requirements
- `BOM_TEMPLATE.md` — bill of materials and cost planning
- `HARDWARE_CHECKLIST.md` — bring-up validation checklist

## Does Not Own

- high-level project vision/roadmap (see `../01_FOUNDATION/`)
- no-hardware simulation execution and tracking (see `../03_EXECUTION/`)

Keep strategy/execution reports outside this folder to avoid overlap.
