# Render Batches

This folder stores render briefs and final outputs by batch ID.

## Structure

- `BATCH-001/BRIEF.md` — first prefilled Omnideck v1 brief
- `BATCH-001/*.png` — final exported renders

## Naming Convention

For each new batch, use:

- `assets/renders/BATCH-XXX/BRIEF.md`
- `BATCH-XXX_01_front_operational.png`
- `BATCH-XXX_02_internal_layout.png`
- `BATCH-XXX_03_field_deployment.png`
